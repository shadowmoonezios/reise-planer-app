# Reise Planer App

Willkommen im **Reise Planer App** Projek!

## Beschreibung
Die Reise Planer App ist eine moderne Webanwendung, die es Nutzern ermöglicht, ihre Reisen effizient zu planen und zu organisieren. Die Anwendung enthält Funktionen zur Verwaltung von Reiseplänen, darunter:

- **Zielort Auswahl** - Nutzer können Reiseziele durchsuchen und auswählen.
- **Aktivitätenplanung** - Nutzer können ihre Aktivitäten für jedes Reiseziel planen.
- **Unterkunftsverwaltung** - Möglichkeit, Unterkünfte hinzuzufügen und zu verwalten.
- **Budgetverfolgung** - Nutzer können ihr Reisebudget überwachen und anpassen.

## Technologien
- Frontend: React.js, Redux
- Backend: Node.js, Express
- Datenbank: MongoDB
- Authentifizierung: JWT (JSON Web Token)

## Erste Schritte
1. Klone dieses Repository.
2. Installiere die Abhängigkeiten mit `npm install`.
3. Starte den Server mit `npm start`.
4. Besuche die Anwendung unter `http://localhost:3000`.
